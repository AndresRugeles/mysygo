Módulos integrados en Python

Python viene con una amplia variedad de módulos integrados que proporcionan funcionalidades listas para usar. Estos módulos cubren diversas áreas, desde manipulación de cadenas y archivos hasta operaciones matemáticas y manejo de fechas.

Ejemplos de módulos integrados

  • math: Proporciona funciones matemáticas como sin(), cos(), sqrt(), etc.
  • datetime: Permite trabajar con fechas y horas.
  • os: Proporciona una forma de interactuar con el sistema operativo.
  • sys: Proporciona acceso a variables y funciones relacionadas con el intérprete de Python.
  • json: Permite trabajar con datos en formato JSON.

Clase datetime

Para importar el módulo y sus clases, se usa: from datetime import [clase]. Por ejemplo: from datetime import date

Definición Ejemplo
Crear una variable de tipo fecha fecha_inicio = date(2025,10,31)
Extraer información de una fecha

Formatear fecha como cadena fecha_str = fecha_inicio.strftime('%Y-%m-%d')

Método datetime
Definición Ejemplo
Crear una variable de tipo fecha y hora, por ejemplo: 31 de octubre 2025 a las 14:30 dt_inicio = datetime(2025,10,31,14,30,0,500000)
Extraer información de una fecha y hora
  • year = dt_inicio.year
  • month = dt_inicio.month
  • day = dt_inicio.day
  • hour = dt_inicio.hour
  • minute = dt_inicio.minute
  • second = dt_inicio.second
  • microsecond = dt_inicio.microsecond
Formatear fecha y hora como cadena dt_str = dt_inicio.strftime('%Y-%m-%d %H:%M:%S')
